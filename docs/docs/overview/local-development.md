# Local Development

Nitric provides a CLI that makes it easy to develop and test your applications locally before going live.

## Prerequisites

You'll need to make sure you have the following installed:

- Docker

## Getting the CLI

### MacOS

With Homebrew

```
brew install nitric
```

Or using curl

```bash
curl ...
```

### Linux

```bash
curl ...
```

### Windows

```
scoop install nitric
```
