Nitric Topics are deployed to Azure using [Azure Eventgrid](https://azure.microsoft.com/en-us/services/event-grid/).

## Azure Resources

The following resources are created when deploying Topics to Azure:

- EventGrid Topic
- Azure Resource Group

## Deployment

During deployment Nitric's CLI builds your Topics will be built as follows:

- The topic definition and resource group are deployed as an EventGrid Topic
- Container apps are subscribed to defined Topics
