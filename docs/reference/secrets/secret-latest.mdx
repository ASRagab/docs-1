Returns a reference to the `latest` version of a secret, regardless of the version's ID.

```javascript
import { secrets } from '@nitric/sdk';

const keyRef = secrets().secret('apiKey');

const latestVersion = keyRef.latest();
```

## Notes

`latest()` is most useful when you always want the most recent secrets values from the secrets manager. Database credentials and API keys are good examples of secrets where the latest value is usually what you want.

For symmetric encryption, you'll need to retrieve the version of the secret used to _encrypt_ a value when you try to _decrypt_ it again. In those cases `latest()` isn't a good choice, use [secret().version()](./secret-version) instead.

## Examples

### Get a reference to the latest secret version

```javascript
import { secrets } from '@nitric/sdk';

const keyRef = secrets().secret('apiKey');

const latestVersionRef = keyRef.latest();
```

### Access the latest value of a secret

```javascript
import { secrets } from '@nitric/sdk';

const keyRef = secrets().secret('apiKey');

const key = await keyRef.latest().access();
```

> See [secret().version().access()](./secret-version-access) for more details.
