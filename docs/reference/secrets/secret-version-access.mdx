Retrieves the value from a version of a secret.

```javascript
import { secrets } from '@nitric/sdk';

const keyRef = secrets().secret('apiKey');

const key = await keyRef.latest().access();
```

## Examples

### Access the latest version of a secret

```javascript
import { secrets } from '@nitric/sdk';

const keyRef = secrets().secret('apiKey');

const key = await keyRef.latest().access();
```

### Access a specific version of a secret

```javascript
import { secrets } from '@nitric/sdk';

const keyRef = secrets().secret('encryptionKey');

const key = await keyRef.version('the-version-id').access();
```
