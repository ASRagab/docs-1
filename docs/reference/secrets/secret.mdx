Creates a reference to a secret in the secrets manager.

```javascript
import { secrets } from '@nitric/sdk';

const keyRef = secrets().secret('apiKey');
```

## Parameters

---

**name** required `string`

The unique name of this secret within the secrets manager.

---

## Notes

Creating a reference to a `secret` won't create the secret in the secrets manager if it doesn't already exist. The secret will be created on the first call to [secret().put()](./secret-put)

## Examples

### Create a reference to a secret

```javascript
import { secrets } from '@nitric/sdk';

const keyRef = secrets().secret('apiKey');
```
